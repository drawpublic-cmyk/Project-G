# Project G

관리데이 앱 MVP 기획 및 디자인 프로토타입입니다.

## 산출물

- `Project G _ 관리데이/index.html`: 모바일 앱형 HTML 프로토타입
- `Project G _ 관리데이/manage-day-mvp.html`: 동일한 HTML 프로토타입 원본 파일
- `outputs/manage-day-product-plan.md`: 관리데이 MVP 기획안

## 핵심 방향

Project G는 네이버예약을 대리 수행하는 서비스가 아니라, 사용자가 네이버예약을 순서대로 직접 확정할 수 있도록 돕는 관리데이 플래너입니다.

예약이 불가능한 경우에는 같은 샵의 다른 시간, 근처 대체 샵, 코스 순서 변경을 제안하는 흐름을 MVP에 포함합니다.
